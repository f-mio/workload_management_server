# workload management

## 本アプリについて
プロジェクトの工数管理を行うためのアプリケーションです。各担当者が、タスクレベルでの工数を日毎に記録できるアプリケーションです。

## 本アプリを用いて実現したいこと
- 期間(月,クォータ等)で工数を見える化し、管理,編集コストを低減させる。

  - プロジェクト単位で工数を見える化する
